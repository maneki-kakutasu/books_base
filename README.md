## Курсовой проект: Интеграционная база данных книг (book24, chitai-gorod, moscowbooks)

Минимально-жизнеспособная реализация для сбора, очистки/дедупликации и поиска книг из нескольких источников.

**Источники данных:**
- ✅ **book24.ru** — основной источник
- ✅ **chitai-gorod.ru** — основной источник
- ⚠️ **labirint.ru** — опциональный источник (временно недоступен из-за проблем с SSL)

### Стек
- Backend: FastAPI + SQLAlchemy (PostgreSQL)
- ETL: Python, Scrapy
- Деплой: Docker / docker-compose

### Структура
- `app` — API, модели ORM
- `scripts` — загрузчики источников и ETL-дедупликация
- `core/import_data` — импорт данных в SQL БД

### Схема БД (упрощённо)
- `products` — канонические книги (после дедупликации)
- `offers` — книги из источников (сырые данные)
- `attributes` — произвольные атрибуты (опционально)
